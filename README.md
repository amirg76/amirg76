### 🌟 Featured project — Kibbutz Kissufim Community Messages App

A platform built after October 7th to help the evacuated **Kibbutz Kissufim** community organize the daily flood of WhatsApp messages — turning an endless chat stream into clear, browsable categories. Built as a pilot by a volunteer team, in collaboration with the AppleSeeds (Tapuach) nonprofit, and covered by *Yediot HaNegev*.

A 3-tier system:

| Repo | What it is | Stack |
|------|-----------|-------|
| [kehilapp-front](https://github.com/amirg76/kehilapp-front) | User-facing web app | React · Vite · Tailwind · Redux Toolkit · React Query |
| [kehilapp-backend](https://github.com/amirg76/kehilapp-backend) | REST API | Node.js · Express · MongoDB · JWT · AWS S3 |
| [kehilapp-admin](https://github.com/amirg76/kehilapp-admin) | Admin dashboard | React · TypeScript · MUI · React Query |
